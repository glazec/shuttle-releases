# Shuttle

Manage AI agent skills, plugins, MCP servers, configs, and sessions across Claude Code, Claude Desktop, Codex, OpenClaw, and OpenCode.

## Download

Download the latest release from the [Releases](https://github.com/glazec/shuttle-releases/releases) page.

### macOS App
1. Download `Shuttle-v*.zip`
2. Unzip and drag `Shuttle.app` to `/Applications`
3. Right-click → Open on first launch (unsigned app)

### CLI
1. Download `shuttle-darwin-arm64`
2. `chmod +x shuttle-darwin-arm64 && mv shuttle-darwin-arm64 /usr/local/bin/shuttle`
3. Run `shuttle --help`

### Auto-Update
- **App**: Settings → Check for Updates
- **CLI**: `shuttle self-update`
